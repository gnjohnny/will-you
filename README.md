# Send Me Ksh 100 😄

A fun and interactive web project built using **HTML, CSS, and JavaScript**.  
The page asks the user if they can send **Ksh 100** with two buttons: **Yes** and **No**.

The twist? When the user tries to hover over the **"No"** button, it moves to a different position, making it almost impossible to click.

## 🚀 Features
- Simple and responsive design
- Interactive button behavior using JavaScript
- Demonstrates DOM manipulation and hover events
- Fun mini project for practicing frontend skills

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
│── index.html
│── style.css
│── script.js


## 🎯 Purpose
This project was created as a **practice project to improve JavaScript DOM manipulation and event handling skills** while building something fun and interactive.

## ▶️ How to Run
1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
